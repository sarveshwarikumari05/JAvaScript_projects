This repository contains a collection of JavaScript mini projects created to strengthen core JavaScript concepts and improve practical problem-solving skills. Each project focuses on real-world functionality and demonstrates how JavaScript can be used to build interactive and dynamic web applications.

About the Repository

The projects in this repository are built using HTML, CSS, and Vanilla JavaScript. They are designed to be beginner to intermediate level and help in understanding important topics such as DOM manipulation, event handling, functions, conditional logic, loops, and basic ES6 features. The code is kept simple, readable, and well-structured to make learning easier.
